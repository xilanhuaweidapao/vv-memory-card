# vv-memory-card

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

### Customize configuration

* 绝对路径还是 @ = src
* public目录的路径是__static
* 要想绑定事件在dom元素上，添加css样式 -webkit-app-region: no-drag
