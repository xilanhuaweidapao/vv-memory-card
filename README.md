
```
npm run build
```

### Run your tests

```
npm run test
```

```
npm run lint
```

### Customize configuration

* 绝对路径还是 @ = src
* public目录的路径是__static
* 要想绑定事件在dom元素上，添加css样式 -webkit-app-region: no-drag 
* 使用正则replace掉####
* browserwindow中设置 webSecurity: false ，不然请求会跨域
* 字体设置为 Consolas 好看很多
